# productive_person (Resource)

Manages a person (user) in Productive.io.

Because Productive.io has no hard-delete endpoint, destroying this resource archives the person (soft delete). Archived persons are treated as destroyed from Terraform's perspective — if a person is archived externally, Terraform will remove it from state on the next refresh.

## Example Usage

```terraform
# Manage a person in Productive.io
resource "productive_person" "example" {
  first_name = "Jane"
  last_name  = "Smith"
  email      = "jane.smith@example.com"
  title      = "Engineer"
  role_id    = 3
}

# A minimal person — only required fields
resource "productive_person" "minimal" {
  first_name = "John"
  last_name  = "Doe"
  email      = "john.doe@example.com"
}
```
