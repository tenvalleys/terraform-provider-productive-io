# Productive Provider

The Productive provider allows you to manage resources in [Productive.io](https://productive.io) using Terraform or OpenTofu.

## Authentication

The provider requires an API token and organization ID. Generate a token at **Settings > API Integrations > Generate new token** in Productive.io.

Credentials can be supplied via the provider block or through environment variables:

- `PRODUCTIVE_TOKEN`
- `PRODUCTIVE_ORGANIZATION_ID`

## Example Usage

```terraform
provider "productive" {
  # Credentials can also be set via PRODUCTIVE_TOKEN and PRODUCTIVE_ORGANIZATION_ID env vars.
  token           = var.productive_token
  organization_id = var.productive_organization_id
}
```
